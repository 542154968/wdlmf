# 关于这个项目
为什么想到写这个项目 说来话长 就是为了方便订单的管理吧 家里亲戚眼睛有点花了 用手机打字不方便
# 功能介绍
```html
获取物品列表 固定数据的JS文件 配合本地存储
将商品列表添加到订单列表中
订单列表可以计算删除
生成文字订单
还有一些本来打算继续开发的 
小程序审核拒绝了 
就没有继续开发的动力了
现在当做测试项目供亲戚使用

```
